* 安装zsh,修改shell后需要exi当前会话
```bash
sudo chsh -s /bin/zsh
```



* 安装 git

* 安装 [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh),安装完后才会有.zshrc配置文件
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
